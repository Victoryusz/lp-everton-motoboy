Política de Privacidade

Última atualização: 05 de setembro de 2025

1. Informações que Coletamos

A Everton Motoboy coleta informações fornecidas voluntariamente pelos usuários através de:

  • Formulários de contato no site
  • Comunicações via WhatsApp, telefone ou e-mail
  • Solicitações de orçamento e serviços

Dados Pessoais Coletados:

  • Dados de identificação: Nome completo
  • Dados de contato: Telefone, e-mail, endereço
  • Dados de serviço: Tipo de serviço solicitado, detalhes da entrega
  • Dados de navegação: Informações sobre o uso do site (cookies)

2. Como Utilizamos suas Informações

Utilizamos os dados coletados exclusivamente para:

  • Prestação dos serviços de motoboy solicitados
  • Comunicação sobre orçamentos e agendamentos
  • Melhoria da qualidade dos nossos serviços
  • Cumprimento de obrigações legais

3. Compartilhamento de Dados

NÃO compartilhamos, vendemos ou alugamos suas informações pessoais com terceiros, exceto:

  • Quando necessário para prestação do serviço (ex: endereço de entrega)
  • Por determinação legal ou judicial
  • Para proteção dos nossos direitos legais

4. Segurança dos Dados

Implementamos medidas de segurança adequadas para proteger suas informações:

  • Armazenamento seguro de dados em sistemas protegidos
  • Acesso restrito apenas a funcionários autorizados
  • Comunicações criptografadas sempre que possível
  • Backup regular dos dados para evitar perdas

5. Seus Direitos

De acordo com a LGPD (Lei Geral de Proteção de Dados), você tem direito a:

  • Acesso: Solicitar informações sobre seus dados
  • Correção: Corrigir dados incompletos ou incorretos
  • Exclusão: Solicitar a exclusão de dados desnecessários
  • Portabilidade: Solicitar cópia dos seus dados
  • Oposição: Opor-se ao tratamento dos dados

6. Cookies

Nosso site pode usar cookies para melhorar sua experiência. Os cookies são pequenos arquivos que:

  • Melhoram a funcionalidade do site
  • Lembram suas preferências
  • Fornecem estatísticas de uso anônimas
